# 🌩️ AWS Website Challenge

## The Challenge

Build and deploy a website on AWS! You have two options:

### Option 1: S3 Bucket Hosting ☁️
Host a static website using Amazon S3. Perfect for portfolios, landing pages, or creative projects.

### Option 2: EC2 Deployment 🚀
Deploy a dynamic website using an EC2 instance. **This option qualifies you for a merch giveaway!** 👕

---

## 📅 Timeline

- **Start**: Today
- **Showcase Deadline**: Tomorrow after 12 PM

---

## 📋 Rules

1. **Choose Your Path**
   - S3 static hosting **OR**
   - EC2 dynamic hosting (required for merch giveaway)

2. **Get Creative**
   - Design matters — make it visually appealing
   - Content matters — have actual meaningful content
   - Structure matters — organize your files properly

3. **Deploy on AWS**
   - Your site must be live and accessible via a public URL
   - Must use either S3 or EC2 (no other hosting platforms)

4. **Submit Your Work**
   - Share your live URL in the showcase channel
   - Deadline: Tomorrow after 12 PM

## ✅ Prerequisites

### Required for Everyone:
- AWS account (free tier available at [aws.amazon.com](https://aws.amazon.com))
- Basic HTML/CSS knowledge

### For EC2 Option (Recommended for merch giveaway):
- SSH client (Terminal on Mac/Linux, PuTTY or WSL on Windows)
- Basic terminal/command line familiarity
- Key pair (.pem file) for EC2 access
